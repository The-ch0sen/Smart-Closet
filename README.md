# Smart Closet Mobile App (Expo)

此資料夾包含 Smart Closet 智慧穿搭助手的 React Native / Expo 專案骨架。專案聚焦於衣物數位化管理、AI 穿搭建議與行為數據學習三大面向，提供後續功能開發的基礎結構。

## 快速開始

> 注意：在此環境中無法預先安裝套件。請於可連線的開發環境執行以下指令完成安裝與啟動。

```bash
cd smart-closet-app
npm install
npm run start
```

## 專案結構

```
smart-closet-app/
├── App.js                  # 應用程式進入點 (目前載入 HomeScreen)
├── app.json                # Expo 組態設定
├── assets/                 # 應用使用的資產 (暫放 1x1 icon)
├── babel.config.js         # Babel 設定
├── package.json            # NPM 套件與腳本定義
├── README.md               # 此說明文件
└── src/
    ├── components/         # 共用 UI 元件 (Header、FeatureCard、ActionButton...)
    ├── constants/          # 常數與假資料 (featureHighlights 等)
    ├── hooks/              # 自訂 hook，模擬每日穿搭推薦
    ├── screens/            # 主要頁面骨架 (Home、Closet、Outfit、Analytics)
    ├── services/           # API 服務模組 placeholder (天氣、衣櫃)
    └── theme/              # 顏色與 spacing 設定
```

# Smart Closet 產品規劃文件

## 1. 產品願景
Smart Closet 是一款專注於個人化穿搭的智慧衣櫥應用，結合衣物數位化管理、AI 穿搭推薦與行為數據學習，協助用戶提升穿衣效率與時尚品味。同時透過虛擬試衣、社交分享與電商導購，打造完整的穿搭生態系。

## 2. 核心價值主張
- 減少「今天要穿什麼？」的決策疲勞，提供即時且多樣化的穿搭靈感。
- 建立雲端同步的個人衣物資料庫，掌握衣櫃使用狀況。
- 結合天氣、行程與個人偏好，生成高度個人化的穿搭建議。
- 將用戶回饋轉化為模型優化素材，持續提升推薦命中率與滿意度。
- 透過社交互動與導購合作，創造穿搭社群與商業價值。

## 3. 主要使用者族群
| 使用者角色 | 需求與痛點 | 主要價值 |
| --- | --- | --- |
| 學生 | 早晨時間有限，需要快速穿搭指引 | 每日推薦、替代方案節省時間 |
| 上班族 | 需兼顧會議、天氣、正式度 | 場景模式與日程整合 |
| 時尚愛好者 | 喜歡嘗試新風格並分享 | 虛擬試衣、社交分享、穿搭收藏 |
| 消費導向用戶 | 期望補足衣櫃空缺與購物建議 | 單品缺口提醒、導購連結 |

## 4. 使用者旅程摘要
1. **建立衣櫥**：拍照或匯入衣物 → 系統自動去背、分類、標籤 → 建立虛擬衣櫃。
2. **每日穿搭**：根據天氣、行程與心情設定 → AI 推薦主穿搭與備選方案。
3. **互動學習**：用戶對推薦按 👍/👎 或調整 → 行為數據回寫模型，優化個人化效果。
4. **社交/導購**：分享今日穿搭、瀏覽朋友 Look → 系統偵測衣櫃缺口並推送購物連結。

## 5. 功能規格
### 5.1 衣物管理
- **衣物上傳**：支援拍照、相簿匯入；背景去除；類別自動標註（上衣/褲子/鞋子/配件）。
- **標籤分類**：顏色、材質、品牌、季節、場合、價格區間、購買日期。
- **虛擬衣櫃**：依類別、標籤、使用頻率進行搜尋與篩選；支援建立收藏夾與搭配套組。

### 5.2 穿搭建議
- **AI 穿搭推薦引擎**：融合天氣、行程、個人風格指標，生成主 Look 與 2 組備選 Look。
- **場合模式**：預設工作、運動、約會、正式場合、旅行等模式，可自訂權重。
- **每日穿搭流程**：早晨提醒 → 查看建議 → 虛擬試衣或套用 → 提供回饋。
- **學習迴圈**：紀錄點讚/點踩、替換單品、穿著時長等資料，持續優化模型。

### 5.3 智能分析
- **天氣整合**：串接 OpenWeather，依據所在地 7 日天氣預測更新建議。
- **衣物使用統計**：顯示出場次數、最近穿著時間、CP 值排序；提供冷門單品提醒。
- **搭配回顧**：依日期、場合或心情標籤瀏覽穿搭歷史，避免重複或撞衫。

### 5.4 社交與導購
- **穿搭分享牆**：支援公開/私密分享、標註單品來源、互動留言。
- **合作導購**：針對衣櫃缺口推播品牌與電商連結，追蹤轉換率。
- **社群挑戰**：舉辦季節挑戰或主題活動，提升參與度。

## 6. 使用者故事
1. 作為一名學生，我希望能快速得到每日穿搭建議，節省時間。
2. 作為一名上班族，我希望系統能根據會議/天氣自動推薦正式穿搭。
3. 作為一名時尚愛好者，我希望能收藏和分享我的穿搭，增加社交互動。
4. 作為一名消費者，我希望當系統發現我缺少某些單品時，可以直接推薦購買連結。

## 7. 技術架構概覽
- **前端（App）**：React Native 或 Flutter，整合影像上傳、推播通知、AR 試衣功能。
- **後端服務**：Node.js + Express/NestJS，提供 RESTful API、排程作業與 WebSocket 推播。
- **資料庫**：PostgreSQL 儲存結構化資料；Redis 作為快取與工作佇列。
- **雲端服務**：Cloudinary/Google Cloud Storage 儲存圖片與去背；OpenWeather 取得天氣資訊。
- **AI 模組**：
  - 影像分類：CLIP / YOLOv8 用於衣物類別與屬性識別。
  - 穿搭推薦：協同過濾 + 規則式 GPT 生成 + 個人化偏好學習。
  - AR 虛擬試衣：Stable Diffusion + 3D Avatar 技術生成試穿視覺。

## 8. 系統模組切分
| 模組 | 主要職責 | 關鍵技術 |
| --- | --- | --- |
| 使用者與身份認證 | 註冊、登入、第三方登入、權限管理 | OAuth 2.0、JWT |
| 衣物資料管理 | 圖片處理、標籤、搜尋、統計 | Cloudinary、PostgreSQL、Elasticsearch (選用) |
| 推薦系統 | 天氣整合、穿搭生成、回饋學習 | OpenWeather API、推薦算法 Pipeline |
| AR 試衣 | 生成個人化虛擬試衣影像 | Stable Diffusion、3D Avatar |
| 社群與導購 | 內容分享、互動、導購追蹤 | GraphQL/REST API、第三方導購 SDK |

## 9. 資料庫概念模型 (草案)
- **User**：帳號資訊、偏好設定、體型資料。
- **Garment**：名稱、圖片 URL、分類、顏色、品牌、材質、季節、場合、標籤。
- **Outfit**：包含多件衣物的搭配組合、適用場合、天氣標籤、使用者回饋。
- **DailyRecommendation**：日期、主 Look、備選 Look、氣象條件、回饋紀錄。
- **Feedback**：推薦結果的評價、調整行為、使用時長。
- **SocialPost**：穿搭分享內容、互動數據、導購連結。

## 10. API 範例 (RESTful)
| 方法 | 路徑 | 說明 |
| --- | --- | --- |
| `POST /api/v1/garments` | 上傳衣物與屬性標註 |
| `GET /api/v1/garments?season=summer&color=blue` | 篩選衣物 |
| `GET /api/v1/recommendations/daily` | 取得今日穿搭推薦 |
| `POST /api/v1/recommendations/{id}/feedback` | 提交推薦回饋 |
| `GET /api/v1/outfits/history` | 查詢歷史穿搭紀錄 |
| `GET /api/v1/social/feed` | 瀏覽社交分享 |

## 12. 成功指標 (KPI)
- 每日活躍用戶（DAU）與留存率（Day-7、Day-30）。
- 推薦採納率（每日推薦 Look 被使用的比例）。
- 用戶回饋互動率（👍/👎、替換單品次數）。
- 冷門單品喚醒率（提醒後 7 日內穿著比例）。
- 導購轉換率與合作品牌營收貢獻。

## 13. 風險與對策
- **資料隱私**：採用資料加密、嚴格權限控管並遵循 GDPR/CCPA。
- **模型偏差**：持續監測推薦公平性，導入人工稽核。
- **影像品質差異**：提供上傳指引與自動修正功能，確保識別準確度。
- **導購信任**：明確標示合作內容，提供用戶回饋管道。

## 14. 未來發展方向
- 開放 API 與第三方穿搭服務串接。
- 與實體服飾品牌合作，提供虛擬試衣到門市購買的 O2O 體驗。
- 引入智慧衣櫃硬體（RFID/IoT）自動同步衣物狀態。
- 擴充為跨平台（Web、智慧手錶）使用體驗。
